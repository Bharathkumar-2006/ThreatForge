Users install the MailCheckr Chrome Extension.

When a user opens an email, the extension extracts sender information.

The extension sends sender data to the MailCheckr Backend API.

The Backend API communicates with IPQualityScore APIs.

The Backend API analyzes:
- Sender reputation
- Fraud risk
- Deliverability
- Domain information

Results are returned to the Chrome Extension.

The extension displays security insights to the user.

MongoDB stores user accounts and API usage information.

JWT is used for authentication.
